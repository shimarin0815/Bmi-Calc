# BMI Calc

身長(cm)と体重(kg)からボディマス指数(BMI)を**自動計算**する、シングルファイルのシンプルな健康チェックSPAです。履歴・アドバイス・LocalStorage保存対応。🧮✨

## 特徴
- 入力: 身長・体重（小数OK）
- 自動計算 & カテゴリ表示（日本肥満学会の区分準拠）
- ゾーン別フィードバック（食事/運動/睡眠/ストレス）
- 計算履歴の保存/再読込/クリア（LocalStorage）
- レスポンシブなモダンUI（ダーク/ライト対応）
- 送信なし：データは端末内のみ
